Projects

PseudoTube

PseudoTube was created using React.js and YouTube API to create a SPA that takes user queries and runs them through an API call that is modified to search for videos. Data from YouTube are compiled into React components and shown on the page. Users are able to watch videos from a list of search result items. A dark mode feature is also available which toggles a darker background and lighter font to create a "theater" like experience.

Created with:

  • React.js
  • Node.js
  • YouTube API
  • Semantic UI

News Vue

News Vue was created using Vue.js and the News-API to create a news aggregate application that takes user queries and runs them through an API call that is modified to search for popular news. Select data is taken from the JSON supplied data and displayed and rendered on the page.

Created with:

  • Vue.js
  • Node.js
  • News-API

Two Bit Budget

Two Bit Budget was created using HTML, CSS and JavaScript to create a user customizable budget tracker. Users can define income and expenses and add a dollar value which is handled by JavaScript and displays whether the user has a positive or negative cash flow.
